Instrucciones para correr el proyecto:

1. Clonar el repositorio
git clone https://github.com/mateocuenya/desarrollo_ws.git
cd ws_desarrollo


2. Instalar dependencias
El proyecto NO incluye la carpeta node_modules (está ignorada en Git).
Por lo tanto, ejecutar:
npm install

3. Ejecutar el servidor
node server.js
Si todo funciona correctamente, el servidor debería indicar que está corriendo, por ejemplo:
Servidor corriendo en http://localhost:3000


4. Abrir la aplicación
Abrir en el navegador:
http://localhost:3000


5. Probar la comunicación en tiempo real
Abrir la aplicación en dos o tres pestañas diferentes.
Iniciar sesión con usuarios precargados.
Enviar mensajes.
Verificar que se visualicen en tiempo real en ambas pestañas.


6.Para detener el servidor
En la terminal presionar:
Ctrl + C
